What is OpenSCHC ?
==================

OpenSCHC
--------

* OpenSCHC is a OpenSource Implementation of SCHC (Static Context Header Compression) : https://datatracker.ietf.org/doc/draft-ietf-lpwan-ipv6-static-context-hc/ for micropython and Python.
